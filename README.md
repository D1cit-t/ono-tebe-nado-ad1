## Описание проекта
Проект представляет собой одностраничный сайт для аукциона вещей, в существование которых никто не верил. Сайт выполнен по макету из Figma и адаптирован для просмотра на различных устройствах.

## Используемые технологии
- **HTML5** - семантическая разметка страницы
- **CSS3** - стилизация и анимация элементов
- **Flexbox и Grid** - построение сеток и адаптивная верстка
- **БЭМ-методология** - организация кода и именование классов
